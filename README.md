# User Management API (NestJS + MongoDB)

Simple REST API with 2 million records seeding and JWT authentication.

## Features

- **Framework:** NestJS (v22+)
- **Database:** MongoDB with Mongoose
- **Auth:** JWT protection for data creation
- **Auto-Seeding:** Generates 2M records on start if DB is empty.
